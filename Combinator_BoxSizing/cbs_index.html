CSS Combinators

A combinator is something that defines the relationship between two or more selectors.

1. Descendant Combinator :    (space)

The descendant combinator matches all elements that are descendants (children, grandchildren, etc.) of a specified element.

CSS

/* Targets every <p> inside .card */

.card p {
    color: red;
    font-weight: bold;
}

HTML

<div class="card" >

 <p>Styled ✓</p>          <!-- direct child -->

 <div>

  <p>Also styled ✓</p>        <!-- grandchild -->

 </div>

</div>

<p>Not styled ✗</p>         <!-- sibling .card -->

OUTPUT

Styled ✓

Also styled ✓

Not styled ✗

1. Child Combinator : > (greater-than)

The child combinator selects all elements that are direct children of a specified element.

CSS

/* Targets Only <p> elements directly inside .card */

.card > p {
    color: blue;
    font-weight: bold;
}

HTML

<div class="card" >

 <p>Styled ✓</p>          <!-- direct child -->

 <div>

  <p>Not styled ✗</p>        <!-- grandchild -->

 </div>

</div>

<p>Not styled ✗</p>         <!-- sibling .card -->

OUTPUT

Styled ✓

Not styled ✗

Not styled ✗

1. Next Sibling : + (plus)

The next sibling combinator is used to select an element that is directly after a specific element.

CSS

/* Targets <p> only if it is the immediate next sibling of .card /

.card > p {
    color: green;
    font-weight: bold;
}

HTML

<div class="card" >

 <p>Not styled ✗</p>          <!-- direct child -->

 <div>

  <p>Not styled ✗</p>        <!-- grandchild -->

 </div>

</div>

<p>Styled ✓</p>         <!-- immediate sibling .card -->

<p>Not styled ✗</p>         <!-- not immediate sibling .card -->

OUTPUT

Not styled ✗

Not styled ✗

Styled ✓

Not styled ✗

1. Subsequent-sibling Combinator : ~ (tilde)

The subsequent-sibling combinator selects all elements that are next siblings of a specified element.

CSS

/* Targets all <p> after .card with same parent /

.card ~ p {
    color: green;
    font-weight: bold;
}

HTML

<div class="card" >

 <p>Not styled ✗</p>          <!-- direct child -->

 </div>

</div>

<p>Styled ✓</p>         <!-- immediate sibling .card -->

<p>Styled ✓</p>         <!-- not immediate sibling .card -->

Box Sizing

The box-sizing CSS property sets how the total width and height of an element is calculated.

CSS Box Model

Everything in CSS has a box around it and every box consists of four parts: content, padding, borders and margins.

Box Model

margin

border

padding

content

box-sizing: content-box

This is the initial and default value as specified by the CSS standard. The width and height properties include the content, but does not include the padding, border, or margin.

CSS 

.box {
    box-sizing: content-box;  /* browser default */
    width: 200px;
    padding: 10px;
    margin: 15px;
    border: 2px solid black;
}

content-box

margin (15+15 = 30px)

border (2+2 = 4px)

padding (10+10 = 20px)

content (200px)

box-sizing: border-box

The width and height properties include the content, padding, and border, but do not include the margin.

CSS 

.box {
    box-sizing: border-box;
    width: 200px;
    padding: 10px;
    margin: 15px;
    border: 2px solid black;
}

content-box

margin (15+15 = 30px)

border (200px)

padding (10+10 = 20px)

content (200-10-10-2-2 = 176px)
